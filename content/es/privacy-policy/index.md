__Resumen__: No usamos cookies y no recopilamos ningún dato personal.

## Visitantes del sitio web

- No se recopila información personal.
- No se almacena información en el navegador.
- No se comparte, envía ni vende información a terceros.
- No se comparte información con empresas de publicidad.
- No se extrae ni recopila información sobre tendencias personales y de comportamiento.
- No se monetiza la información.

### Información que recopilamos y para qué la usamos

Ejecutamos [Plausible](https://plausible.io/) analytics en getdoks.org. Se recopila la siguiente información:

- __URL de la página__. Rastreamos la URL de cada vista de página en este sitio web. Usamos esto para entender qué páginas han sido visitadas y cuántas veces se ha visto una página en particular. Por ejemplo: _https://getdoks.org/_.
- __Referente HTTP__. Usamos la cadena de referente para entender el número de visitantes referidos a este sitio web desde enlaces en otros sitios. Por ejemplo: _https://github.com/_.
- __Navegador__. Usamos esto para entender qué navegadores usan las personas al visitar este sitio web. Esto se obtiene del encabezado HTTP User-Agent. El User-Agent completo se descarta. Por ejemplo: _Chrome_.
- __Sistema operativo__. Usamos esto para entender qué sistemas operativos usan las personas al visitar este sitio web. Solo usamos la marca del sistema operativo y no incluimos el número de versión ni ningún otro detalle. Esto se obtiene del encabezado HTTP User-Agent. El User-Agent completo se descarta. Por ejemplo: _GNU/Linux_.
- __Tipo de dispositivo__. Usamos esto para entender qué dispositivos usan las personas al visitar este sitio web. Esto se obtiene de window.innerWidth. El ancho real del navegador en píxeles se descarta. Por ejemplo: _Desktop_.
- __País del visitante__. Buscamos el país del visitante usando la dirección IP. No rastreamos nada más granular que el país de origen y la dirección IP del visitante se descarta. Nunca almacenamos direcciones IP en nuestra base de datos o registros. Por ejemplo: _Canadá_.
