# Coraza.io Website

The official website for [OWASP Coraza Web Application Firewall](https://coraza.io), built with [Hugo](https://gohugo.io/), Bootstrap 5, and vanilla JavaScript.

## Prerequisites

- [Hugo extended](https://gohugo.io/getting-started/installing/) (v0.80.0 or later)
- [Node.js](https://nodejs.org/) (v16 or later)
- [Go](https://golang.org/) (v1.21 or later)

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm start

# Build for production
npm run build
```
