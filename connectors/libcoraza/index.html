libcoraza (C Library)

Welcome to libcoraza, the C library for OWASP Coraza Web Application Firewall. Because Coraza was made in golang, if you want to embed in any kind of C application, you will need this library.

Prerequisites#

  • a C compiler:
    • gcc or
    • clang
  • Golang compiler v1.24+
  • libtools
  • autotools
  • make
